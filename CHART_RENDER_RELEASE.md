# AdofaiTweaks — 谱面视频渲染 正式发布

《冰与火之舞》游戏内离线录屏工具，不录桌面、不录 UI，成品视频可直接投稿。

---

## 核心特性

- **纯净画面** — 只录制谱面轨道和游戏画面，不包含编辑器按钮、UMM 界面、鼠标指针
- **固定帧率** — 设置多少 fps 就输出多少 fps。电脑慢只影响等待时间，不影响视频时间轴
- **音画同步** — 音频通过 Unity mixer 离线捕获，原曲 + 打拍音 + 长按音效 + PlaySound 全部保留；视频与音频无相位差
- **多格式输出** — MP4 / MKV / MOV，音频 AAC / FLAC / ALAC
- **智能码率** — 默认自动推荐（1080p→20M, 2K→35M, 4K→60M），也可手动指定
- **硬件加速** — 自动检测 NVENC GPU 编码，回退 CPU x264 兜底
- **预览模式** — 完整 / 暗淡 / 关闭三档，渲染时可选透明浮窗

## 设置面板

浮窗内可直接调：分辨率（1K/2K/4K 快捷预设）、帧率（30/60/120）、CRF 画质、尾巴秒数、显示判定、音频偏移、码率模式、编码器、预览模式、输出/音频格式、专业 FFmpeg 参数

## 使用方式

1. 启用「谱面视频渲染」
2. 打开关卡进入编辑器
3. 调好参数，点「开始渲染」
4. 渲染中可点「取消渲染」或按 ESC 中断
5. 完成后视频在导出目录

## 录制原理

逐帧锁死游戏时间轴 → AsyncGPUReadback 捕获画面 → FFmpeg 编码 → 合成音频 → 输出文件

## 注意事项

- 需要 FFmpeg（会自动按「系统 PATH → Tools 文件夹 → 默认路径」查找，或手动放入 `Mods/AdofaiTweaks/Tools/ffmpeg.exe`）
- 4K 渲染内存峰值约 2GB，建议 8GB+ 内存
- 片段渲染（「仅渲染选中段落」）计划在后续版本加入
